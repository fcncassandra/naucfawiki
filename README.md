# 项目概述

[返回目录](#目录)

知识库计划是南审CFA编辑部发起的纯公益学术性项目，该项目将通过调研采访等方式来收集金融行业各职业发展方向所必备的相关技能和学习资料，帮助南审CFA在校生接触到真实的市场情况和职业技能需要，为自己的职业发展早做规划。[详细](./项目介绍.md)
